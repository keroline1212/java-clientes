# 💻 Sistema de Cadastro de Clientes em Java

## 📌 Sobre o projeto

Este é um projeto simples desenvolvido em Java com o objetivo de praticar conceitos básicos de programação, como:

* Criação de classes
* Organização em pacotes
* Estrutura de um sistema simples
* Uso de lógica de programação

## 📁 Estrutura do projeto

```
src/
 ├── clientes/
 │    ├── Cliente.java
 │    └── Main.java
 ├── empresa/
 │    ├── MediaAluno.java
 │    └── Outros exercícios
```

## ⚙️ Tecnologias utilizadas

* Java
* Eclipse IDE

## ▶️ Como executar

1. Clone o repositório:
```
git clone https://github.com/keroline1212/java-clientes.git
```

2. Abra no Eclipse

3. Execute a classe `Main.java`

## 🎯 Objetivo

Praticar programação em Java e entender como organizar um projeto utilizando boas práticas.

## 📚 Aprendizados

* Uso do Git e GitHub
* Organização de código em pacotes
* Estrutura básica de um sistema em Java
